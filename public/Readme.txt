Bosse Quick-Order

Projektbeschreibung

Bosse Quick-Order ist eine webbasierte Anwendung, die Kunden ermöglicht, schnell und bequem Produkte aus verschiedenen Kategorien zu bestellen. Die intuitive Benutzeroberfläche unterstützt dynamische Produktauswahl, automatische Preisberechnung und ist responsiv gestaltet, sodass die Anwendung auf allen Endgeräten gut nutzbar ist.

Dieses Projekt dient insbesondere dazu, die Bestellabwicklung bei Getränkebestellungen effizienter zu gestalten und den Kunden eine benutzerfreundliche Plattform bereitzustellen.

⸻

Hauptfunktionen
	•	Dynamische Produktauswahl: Auswahl von Produkten basierend auf Kategorie und Produkttyp.
	•	Intelligente Dropdowns: Optimierte Dropdown-Menüs (mit Select2) zur leichteren Bedienbarkeit.
	•	Mengen- und Preiskalkulation: Automatische Berechnung von Zwischen- und Gesamtpreisen.
	•	Behälter- und Volumenauswahl: Auswahl zwischen Flaschen (0,5 l, 0,75 l, 1 l) und Kanistern (5 l, 10 l).
	•	Responsives Design: Anpassung der Benutzeroberfläche an verschiedene Bildschirmgrößen.

⸻

Technischer Stack
	•	Frontend: HTML5, CSS3 (inklusive Bootstrap 5), JavaScript
	•	Bibliotheken: jQuery, Select2
	•	Versionierung: Git

⸻

Installation und Lokale Nutzung
	1.	Klone das Repository:

git clone https://github.com/xDima87x/bosse-quick-order.git
cd bosse-quick-order

	2.	Installiere nötige Abhängigkeiten:

npm install

	3.	Starte den lokalen Webserver:

npm start

       Der Express-Server läuft über `node index.js` und bedient das Verzeichnis `public`.

	4.	Öffne die Webseite in deinem Browser unter:

http://localhost:3000
